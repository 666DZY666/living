### 零售的哲学

    书籍1 7-Eleven便利店创始人自述
    https://book.douban.com/review/9603973/
    
    经营理念：
      1. 真正的竞争对手不是其他品牌的便利店，而是不断变化的“客户需求”。
      2. 专注于地方居民的需求变化，打造最优化的门店，全力提高生产效率和产品价值。
      
    营业额远高于其他连锁便利店的原因：
      1. 始终贯彻密集型选址战略；
         优势：一定区域内加深品牌认知度、提升物流和配送效率、广告和促销宣传更有成效。
      2. 具备产品研发与供应的基础体系；
         产品的研发能力已成为现代企业的核心竞争力。
      3. 注重与员工的直接沟通；
         店铺经营顾问，是7-Eleven的独特职位，是便利店事业发展道路的重要支柱。
