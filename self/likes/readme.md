技术类工作十余年，思维方式愈发趋于**纯理性动物：万物有定义、有果必求因、凡事靠推理**，诚然这样的方法论太过**缺乏人文情怀**。不光自己，身边的大批科学家、工程师也深受纯粹理性的毒害。加之在匪帮的体制下从小到大都没有接受过正式的音乐教育，导致绝大多同学根本不懂音乐，也不会欣赏音乐，实乃人生一大憾事。
